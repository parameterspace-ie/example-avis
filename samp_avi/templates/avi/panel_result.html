Result view help

This is a panel which renders one of the job results.

Select an area of the plot to zoom in on it.

  • The data is rendered using highcharts.
  • The pipeline creates a large JSON object using two columns from the resulting VOTable from the query.
  • When a result is accessed, javascript makes a call to an AVI endpoint which returns the JSON data for a particular job.
  • This data is then dynamically loaded in to the highchart object.
