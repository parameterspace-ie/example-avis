Multiple Pipelines AVI Help

This AVI which demonstrates the following:

  • Builds on Simple AVI
  • Contains multiple AVI pipelines
    1. Demonstrates prototype of running (DPAC) Java tools in AVI pipeline, using the CU8 Ulysses BP/RP Simulator.
    2. Analysis of GACS-Dev IGSL source samples that were originally in OGLE-III.
  • Note: this AVI should only be used with the latest repositories.gavip.science/ps_avi_java_7 container template image, with the Ulysses tool being mounted into /opt/ulysses as follows. See the Developer Guide for details.

Below is a list of some of the important files in this AVI:

Filename Description Related
models.py Defines the database models which store the job parameters. The parameters stored in these models are used in the pipeline. tasks.py
tasks.py Defines the AVI pipelines. models.py
urls.py Defines the URLs of the AVI. views.py
views.py Defines functions which handle HTTP requests. tasks.py
