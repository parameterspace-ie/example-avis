Result view help

This is a panel which renders one of the job results.

Toggle the variable details to see more information about the results.

  • The data is rendered using Pandas and Pandas-profiling.
  • The pipeline creates a large JSON object using several columns from the resulting VOTable from the query.
  • Pandas is used to create the HTML that presents the data. This is stored in the output of the job.
  • When a result is accessed, the HTML for a particular job is passed in as context to the rest of the results template.
